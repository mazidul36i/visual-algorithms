# Visual Algorithms

Visual Algorithms is a high-performance, visually immersive showcase for interactive Data Structures & Algorithms and mathematical visualizations. It transforms complex interactions and logic into beautiful, interactive art.

## Features

- **Sorting Visualizer**: Interactive visualizations for Bubble Sort and Merge Sort.
- **Flow of Pi**: A unique visualization exploring the digits of Pi.
- **Modern UI**: Built with a "glassmorphism" aesthetic, featuring smooth animations and dark mode.

## Tech Stack

This project is built with modern web technologies:

- [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling
- [React](https://reactjs.org/) - A JavaScript library for building user interfaces
- [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript at Any Scale
- [Tailwind CSS](https://tailwindcss.com/) - Rapidly build modern websites without ever leaving your HTML
- [shadcn/ui](https://ui.shadcn.com/) - Beautifully designed components built with Radix UI and Tailwind CSS

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   cd VisualAlgorithms
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
